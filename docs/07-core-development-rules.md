# Genzo-Kit Core Development Rules

Permanent and non-negotiable.

1. **Structure**: Root = `genzo-kit`. Tools in `src/tools/[kebab-case]`. Never modify existing tools for new features.
2. **Docs**: After ANY change, update ALL 8 docs files.
3. **Workflows**: New feature → `05`. Bug fix → `06`.
4. **Tech**: Tauri v2, React, TypeScript, TailwindCSS. `genzo-kit.exe`. RAM < 60 MB, startup < 0.6s.
5. **UI**: Dark theme only. Professional IDE-like look.
6. **Code**: Strict TypeScript. User-friendly error handling.
7. **Testing**: Verify Text Comparator works after every change.
8. **Naming**: Genzo [Tool Name].
